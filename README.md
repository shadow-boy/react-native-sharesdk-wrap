# react-native-sharesdk-wrap

social share module with sharesdk native

## Installation

```sh
npm install react-native-sharesdk-wrap
```

## Usage

```js
import SharesdkWrap from "react-native-sharesdk-wrap";

// ...

const result = await SharesdkWrap.multiply(3, 7);
```
